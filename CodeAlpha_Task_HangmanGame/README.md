#  Hangman Game (Python)

##  CodeAlpha Internship – Task

This project is a **simple text-based Hangman game** developed using Python as part of the **CodeAlpha Internship – Task **.  
The game allows the player to guess a randomly selected word one letter at a time with a limited number of incorrect attempts.

---

##  Project Description

Hangman is a classic word-guessing game.  
In this implementation:
- A word is randomly chosen from a predefined list
- The player guesses one letter at a time
- The player has a maximum of **6 incorrect guesses**
- The game ends when the word is guessed correctly or attempts are exhausted

This project focuses on **basic Python programming concepts** and **console-based interaction**.

---

##  Features

- Console-based game (no graphics)
- Random word selection
- Maximum 6 wrong attempts
- Displays progress after each guess
- Beginner-friendly logic
